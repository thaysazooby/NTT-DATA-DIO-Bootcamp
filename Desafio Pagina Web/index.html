Inspetor de elementos

(Voltar)

A ferramenta de inspecionar elementos auxilia a visualização do código-fonte de uma página web e através dela é possível testar alterações temporárias e avaliar os efeitos das mudanças sem que isso altere o código.

Estrutura Básica do HTML

(Voltar)

As sessões básicas que constituem uma página HTML são: html, head e body.

  1. HTML: essa tag abre e fecha uma página web
  2. HEAD: o cabeçalho da página, contém informações necessárias para o navegador e outros serviços web, não fica visível para o usuário.
  3. BODY: contém o corpo da página, tudo que será exibido ao usuário, o que fica visível na janela de navegação.

Tags

(Voltar)

As tags delimitam as estruturas da página web. Algumas tags possuem atributos que podem ser ativados proporcionando uma manipulação da estrutura desenvolvida (ex: id, style, class, type, src...) Aqui estão algumas tags vistas na aula:

  • Input: gera uma caixa de texto a ser preenchida pelo usuario.
  • Imagem - img: permite adicionar uma imagem na pagina web.
  • Texto: existem variadas tags para estruturar os textos como as tags de titulos que alteram os tamanhos das letras (h1, h2, h3, ate h6), existem tambem as tags que definem os textos, os estilos, paragrafos, quebras de linha (p, span, i, strong, hr, mark, small, sub, sup, blockquote)
  • Listas - ul/ol: as tags de listagem são de dois tipos, as ordenadas com números (ol) e as não-ordenadas, cada novo tópico listado deve ser introduzido com a tag li. (ul)
  • Links: são as chamadas tags de ancoragem, que estabelecem conexões de uma determinada parte da página com outro trecho, ou uma nova guia, ou ainda uma nova janela de navegação através do link inserido nela.
  • Tag extras - font, del, abbr: A tag "font" permite realizar alterações estéticas/visuais no texto, tais como cor, tamanho e estilo. "Del" é usada para identificar um trecho que foi deletado do texto, mas permite visualizar o histórico de alterações realizadas no documento. Já "abbr" serve para inserir o nome completo de uma sigla abreviada no texto.
